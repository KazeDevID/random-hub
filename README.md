# Random Hub

random nsfw anime & real

## Intallation ?
- In the browser
```html
<script type="text/javascript" src="./random-hub.js"></script>
```
- NPM
```js
npm i @kazesolo/random-hub
```

## How to use ?
- In browser
```js
const hub = new RandomHub();
console.log(hub.getRandomHub());
```
- In node.js
```js
const RandomHub = require('random-hub').RandomHub;
const hub = new RandomHub();
console.log(hub.getRandomHub())
```
